Browser Extension
Send pages and selections from Chrome/Edge directly to PulseKeeper

🧩 Install the Extension

  1. Open chrome://extensions (Chrome) or edge://extensions (Edge)
  2. Enable Developer mode
  3. Click Load unpacked
  4. Select the extension folder inside the PulseKeeper directory

⚡ How to Use

  • Right-click any page → "Send page to PulseKeeper"
  • Select text, right-click → "Send selection to PulseKeeper"
  • Right-click any link → "Send link to PulseKeeper"
  • Click the extension icon to send the current page

🔌 Capture Server Status

Checking…

PulseKeeper runs a local server on http://localhost:7828 that the extension connects to. Make sure PulseKeeper is running when using the extension.
